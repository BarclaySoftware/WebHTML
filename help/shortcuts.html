Keyboard Shortcuts

Below is a table of all keyboard shortcuts available in WebHTML.

Shortcut Usage
CTRL+A Select all text in the editor
CTRL+C Copy the selected text to the clipboard
CTRL+D Select all instances of a text
CTRL+F Search for text in the editor
CTRL+G Go to a line
CTRL+H Search for text in the editor
CTRL+I Open the suggestions menu
CTRL+L Select the current and subsequent lines
CTRL+O Open an new HTML document
CTRL+S Save the current HTML document
CTRL+U Select all text
CTRL+V Paste text from the clipboard
CTRL+Y Redo
CTRL+Z Undo
ALT+C Highlight all instances of selected text
ALT+M Toggle the minimap
ALT+Z Toggle linewrap
